### Funcionalidad Adicional
Luego de cambiar el color de pantalla del Frame sin importar que boton se aplaste, 
se procede a realizar un log de los cambios realizados a la pantalla dentro de un archivo plano de texto.
Esto fue realizado usando el patron de diseno Observer y Subject. Observer y Subject implementados dentro del Aspecto para hacer uso de la nueva funcionalidad compartida por todos los botones (cambiar color).
